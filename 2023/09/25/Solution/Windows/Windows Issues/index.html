Debris

  • Windows 7/10 桌面壁纸保存路径：%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Themes\CachedFiles\
  • C:\ProgramData\Package Cache 中存有应用的安装程序，这些安装程序被 Windows 应用管理器用于卸载应用，因此请勿随便修改。

Solution

时间乱码

电脑部分软件时间出现乱码，如：〉〇〉《-〇【-〉》 〉《:〉『:》《 ，场景包括”phpStudyPro”与取证软件恢复的数据。

后来发现是 Windows 在『区域格式』设置为『中文（简体汉字，香港特别行政区）』时，解析部分软件的时间数据可能会出现乱码

解决办法：在 Windows设置/时间和语言/区域 下设置『国家或地区』为中国，『区域格式』为『中文（简体，中国）』

ext4.vhdx

WSL2-Kali-linux 遗留下来的虚拟磁盘文件”ext4.vhdx”文件，想在 Windows 中访问有两种方式：

  1. 直接用 7z 解压查看，即可看见完整的文件系统目录，文件可完整读取
  2. Win11 特定版本及以上，WSL 提供了 wsl --mount 命令可用于挂载虚拟磁盘。（未实践，纯脑测）
    1. 首先在『磁盘管理/操作/附加 VHD』中添加 vhdx 磁盘
    2. 在 pwsh 中使用 GET-CimInstance -query "SELECT * from Win32_DiskDrive" 命令获取 windows 可读取的外部磁盘的识别路径，如 \\.\PHYSICALDRIVE1
    3. 使用 wsl --mount "\\.\PHYSICALDRIVE1" 挂载磁盘
    4. 进入 wsl，用 sudo mkdir /mnt/kali 创建挂载点
    5. 使用 ls /dev 查看挂载的磁盘对应标识符，例如为 /dev/sde
    6. 挂载磁盘：mount /dev/sde /mnt/kali

Errors

  1. 运行某 GalGame 的 EXE 文件时报错：

    可能原因：

    1. 缺少微软的 VS 运行库：Microsoft Visual C++ 2008 Redistributable Package (x86) 微软官网下载
      下载中文版 64 位安装程序，运行安装后重启生效，EXE 文件可用。

    2. win+r，输入 services.msc ，检查并打开『Windows Modules Insatller』服务。

      （上述方法经测试并无效果，仅作留档参考）

    题外话：使用 sxstrace.exe 跟踪 Windows 错误日志。

    该工具可保存一闪而过的”Windows Error”的具体信息。
