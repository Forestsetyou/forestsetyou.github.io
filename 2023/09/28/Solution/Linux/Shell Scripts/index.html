变量

  • 变量名：字母、数字、下划线，数字不可开头

  • 变量赋值：

    字符串常量可以不带双引号，连接字符串不用 + （会被识别为字符串内容）

  • 变量引用：

    双引号包裹可以避免很多奇怪的错误，例如它会解析变量中的换行与换行符。

普通数组

关联数组

  • @, * 可以获取数组中的所有元素

  • ${#arr[@]} 获取数组长度

  • ${arr:2:3} 截取数组元素，从下标 2 开始截取 3 个
